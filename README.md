# 🌱 Smart Plant Monitoring System

A smart IoT-based plant monitoring and irrigation system built using ESP32. This project monitors soil moisture, temperature, humidity, and human presence in real time while providing automated watering and remote monitoring features.

## 🚀 Features

* Real-time soil moisture monitoring
* Automatic water pump control using relay
* Temperature and humidity monitoring with DHT11
* OLED display for live sensor readings
* Human detection alerts using IR sensor
* Wi-Fi connectivity using ESP32
* Telegram bot notifications
* Remote monitoring and control through phone/web interface

## 🛠️ Components Used

* ESP32 Dev Board
* Soil Moisture Sensor
* DHT11 Sensor
* Relay Module
* Mini Water Pump
* OLED Display
* IR Sensor
* Buzzer
* LEDs
* Jumper Wires & Breadboard

## 💡 Applications

* Smart gardening
* Home automation
* Indoor plant care
* Automated irrigation systems

## 📷 Project Highlights

This project combines embedded systems, IoT, automation, and PCB design into a single practical application. The system is designed to help users monitor plant health efficiently while reducing manual effort.
